# Talha Tool: Android App
A mobile companion app that allows technicians to diagnose iPhones on the go, delivering instant repair solutions without needing a PC. Download `TalhaTool.apk` to install on your Android device.

**Key Features:**
* **Camera Scan:** Use your phone's camera to scan a panic log directly from a broken iPhone's screen and get an instant result.
* **Import Panic Log File:** Select and import a saved panic log file from your phone's storage.
* **Input Code Manually:** Type a panic code manually to look up the failing hardware.
* **Paste Panic Log:** Paste the copied text of a full panic log to run a diagnosis.
