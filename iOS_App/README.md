# Talha Tool: iOS App
The native iOS version of the diagnostic tool, providing technicians with powerful on-the-go log scanning capabilities. Download `TalhaTool.ipa` to sideload onto your iOS device.

**Key Features:**
* **Camera Scan:** Use your iPhone's camera to scan a panic log directly from another device's screen and get an instant result.
* **Import Panic Log File:** Select and import a saved panic log file directly from your iOS files.
* **Input Code Manually:** Type a panic code manually to look up the failing hardware.
* **Paste Panic Log:** Paste the copied text of a full panic log to run a diagnosis.
