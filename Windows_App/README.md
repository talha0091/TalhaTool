# Talha Tool: Windows App
A powerful desktop utility that connects to an iPhone via USB to automatically diagnose hardware faults. Download `talhatoolv5.exe` to run the tool on Windows 10 or 11.

**Key Features:**
* **Run Diagnostic Scan:** Automatically extracts the panic log file from a connected iPhone, runs the diagnosis, and displays the result.
* **Import Panic Log:** Allows users to import a saved panic log file directly from their computer to get a result.
* **Input Panic Code Manually:** Users can manually type in a specific panic code to get an instant hardware diagnosis.
* **Export:** Allows users to export and save the extracted panic log files and diagnostic reports.
* **Install on Android:** Connect an Android phone to install the mobile companion app directly from the desktop tool.
* **Install on iPhone:** Connect an iOS device to securely sideload the companion app directly to the iPhone.
